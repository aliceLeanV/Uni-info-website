    Accessibility

    Font colours and size

    The webiste being distinguishable is important to make it eaier for users to see content including separating foreground from background (w3c, 1.4). The website makes use of colours contrasts, such as the white textbox on a darker colour, to improve the readability for users with any visual impairments. The font size is no smaller than 15px and are in a consistent, readable font throughout.

    The quiz on the website uses multiple cues, colour and text, to indicate the answers. This is to ensure users who may be colour blind or be using any colour-blindness filter.

    The website makes use of clear navigation, at the top of the page through the use of a hamburger menu and bottom of the page, to ensure proper structure and therefore, the user can navigate across the page easily with the appropriate labels. Additionally, when the user hovers over a link, the font becomes bold so the user is aware which link they are clicking.

    Accessibility handled by other softwares

    If a user uses a zoom tool on the website, whether to see the text or images clearer, the website will maintain its structure to ensure a clear layout.

    If a user has a visual impairment and wishes to use another software such as a screen reader when using the website, the images on the website makes use of alternate text, so users who are unable to view the images can receive a description of it.
