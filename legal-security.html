    Legal and security

    Copyright law

    Used images have been cited and referenced in harvard style, underneath each page with the name, date if available, title of the image/code and the date accessed, to adhere to the copyright law. The links take the user to the webpage where the image was found.

    The same has been done with the code used to create the hamburger menu. It has been referenced on the page and in the code, html and css.

    GDPR

    When the user enters their email, there is a note mentioning their email will not be shared to third parties without the users consent. This protects the users personal data, following the GDPR guidelines.
